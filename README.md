# 🚀 Flotas SC 1.0 - Star Citizen Fleet Manager

![Star Citizen Inspired UI](https://img.shields.io/badge/UI-Star_Citizen_Inspired-00d5ff?style=for-the-badge)
![Version](https://img.shields.io/badge/Version-1.0_Production-green?style=for-the-badge)
![OS](https://img.shields.io/badge/OS-Windows-blue?style=for-the-badge)

**Flotas SC** es una herramienta de gestión de flotas para Star Citizen diseñada con una estética futurista de alta tecnología. Permite a los ciudadanos organizar sus naves, crear listas de favoritos y sincronizar prefijos directamente con el archivo `global.ini` del juego para una identificación rápida in-game.

## ✨ Características Principales

- **🎨 Interfaz Ultra-Moderna**: Inspirada en el HUD de las naves de Star Citizen, con una paleta de colores cian, ámbar y negro profundo.
- **⭐ Sistema de Favoritos**: Crea listas personalizadas de naves que aparecerán primero en tu inventario del juego.
- **🔢 Numeración Inteligente**: Clasificación automática (1., 2., 3...) para una organización visual perfecta.
- **🔍 Buscador Avanzado**: Encuentra cualquier nave de la base de datos de Star Citizen al instante.
- **🔄 Sincronización Directa**: Escribe los cambios en el archivo `global.ini` de forma segura.
- **📦 Portable**: Un único archivo `.exe` sin necesidad de instalación.

## 🛠️ Instalación y Uso

1. Descarga el archivo `Flotas SC 1.0.exe` desde la sección de [Releases](https://github.com/SigmundBathory/Flotas-SC/releases).
2. Ejecuta la aplicación.
3. Selecciona la carpeta raíz de tu instalación de Star Citizen (ej: `C:\Program Files\Roberts Space Industries\Star Citizen\LIVE`).
4. ¡Empieza a organizar tu flota!
